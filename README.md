# Edutech Platform Design

Бұл жоба Edutech (білім беру) саласына арналған платформа дизайны.
Технологиялар: React, TypeScript, Vite, Tailwind CSS.

## Қалай іске қосу керек?
1. Репозиторийді клондап алыңыз: `git clone <URL>`
2. Папкаға кіріңіз: `cd <папка_аты>`
3. Тәуелділіктерді орнатыңыз: `npm install`
4. Жобаны іске қосыңыз: `npm run dev`
